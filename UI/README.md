Hugo App
## Project Setup

### Installing

First clone project and install dependencies

```sh
$ mkdir hugoapp && cd hugoapp
$ git clone https://github.com/ilkeraltin/hugoapp.git
$ npm install
```

Run on local

```sh
$ npm run dev
```

## Deployment

Deployment build

```sh
$ npm run build:prod
```


```sh
npm install @babel/core @babel/node --save-dev
npm install @babel/preset-env --save-dev
```
